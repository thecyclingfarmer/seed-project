# Getting started

```bash
git clone https://github.com/thecyclingfarmer/seed-project.git
cd seed-project
bower update
npm install
```
# Create .env file
Change paths to fit your project

```bash
BUILD_PATH=build/
MOCK_LAYOUT=app/framework/*
MOCK_PAGES=['app/pages/**/*.html']
MOCK_COMPONENTS=['app/blocks/**/*.html', 'app/components/**/*.html']
```
